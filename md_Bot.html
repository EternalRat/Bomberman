Bomberman
Bot documentation

A bot in this bomberman is using a lua script as way to choose what to do So you can make your own with some helper functions given from C++

Update function

Each frame, the game will call the "Update" function in the script. This is the same lua state as the last call so this means you can set global variables to keep data between frames

Update function should take no arguments.

Map Blocks

nothing = 0
